Personal Finance Calculators

The Simple Dollar is an personal financial advice website. When they came to us, they wanted an embeddable widget for calculating debt payoff schedules. We've now built them almost a dozen calculators of all sorts, including federal tax rates, state tax rates, auto loan calculations, and expected baby and pet costs.

We researched the financial algorithms and rates, as well as wrote the design and code all from scratch.

Technically, the project uses the Mithril.js and D3 libraries to generate the live charts and graphs, and our own in-house CSS layout library for the styling.
