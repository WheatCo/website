Case Study: Real-Time Web Apps

We've been seeing a increasing demand for webapps that update in real-time with the server in response to incoming data or user input. These are apps that behave similarly to Trello, where users don't have to press a "Submit" button to see changes update instantly across several computers. Real-time apps result in massive gains in productivity and collaboration.

Our engineers have written backend code in Python, Ruby, Rails, PHP, Scala, and Rust. However, most of our real-time projects are implemented in Node using the Feathers.js library.

While none of our real-time projects are currently public, we've implemented numerous, and have developed custom tooling to speed up their development.
